## [v1.5] - 2026-04-27

### Catatan
Ini adalah baseline commit pertama ke Git. Semua file yang ada saat ini merepresentasikan kondisi HadirquGO versi 1.5.

### Fitur yang Sudah Ada (Baseline)
- Sistem Attendance (QR Code, GPS-based)
- Gamifikasi: Points, Levels, Achievements, Leaderboard
- Dashboard Lecturer & Student
- Team Management & Team Leaderboard
- Quiz & Super Quiz System
- Challenge System
- Daily Check-in
- Reward & Redeem System
- Calendar & Attendance History
- Viewboard & Public Viewboard
- Business/Owner Module (Staff, Products, Banners)
- Admin User Management
- Social Authentication (Google OAuth)
- Ranking System (Daily, Weekly)
- Student Evaluation
- Testimony System
- Feedback System
- Journey Tracking
- Artisan Commands: Leaderboard calculation, Achievement assignment, Statistics update, dll.
- Whitepaper dokumentasi

### Tech Stack
- Laravel (PHP)
- MySQL
- Blade Templates
- Bootstrap / Tailwind CSS
- Google OAuth Integration
